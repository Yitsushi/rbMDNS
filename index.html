rbMDNS by Yitsushi

ruby MPD-DBus Notify System

This is a "GateWay" between MPD and DBUS...
Currently only supported by Kopete and Pidgin, but most applications are in process.

Dependencies

  • DBus [http://www.freedesktop.org/wiki/Software/dbus]
  • MPD [http://mpd.wikia.com/wiki/Music_Player_Daemon_Wiki]
  • ruby [http://www.ruby-lang.org/]
  • ruby-dbus [https://trac.luon.net/ruby-dbus/]
  • librmpd [http://librmpd.rubyforge.org/]
  • YAML [http://ruby-doc.org/core/classes/YAML.html]
